CSS BOX MODEL

  1. What is a box model?

    2. A box model is the box that grows around various HTML elements in a web page.


  2. What are the properties of a box model?(Not less than 5)

    3. The properties of a box model are:

    • Margin ~ is a transparent area that goes around all the properties of a box model.
    • Border ~ is the area that goes around the padding and content.
    • Padding ~ is the area around the content.
    • Content ~ is where the text and images appear.
